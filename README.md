# Personality Collection — Frontend

A React-based survey platform that assesses YouTube creators' personality types and delivers AI-generated growth recommendations.

## What It Does

YouTubers receive an email invitation with a unique link. They complete a personality survey (Big 5 model), confirm channel ownership, and get a personalised report with creator type, strengths, weaknesses, and a growth strategy — all generated by AI.

## Tech Stack

- **React 18** — functional components, hooks
- **React Router 6** — client-side routing
- **TanStack React Query** — server-state caching
- **Redux Toolkit** — global loading state
- **Axios** — HTTP client with interceptors
- **Tailwind CSS** — utility-first styling, dark mode
- **Ant Design** — modals, inputs, spinner

## Project Structure

## Running Locally

### Prerequisites

- Node.js 16+
- npm or yarn
- Backend API running on `http://localhost:9718` (see backend repo `https://github.com/Alex-skigo99/API-Unlock-Growth-Strategy`)

### Setup

```bash
# Install dependencies
npm install

# Start development server (default: http://localhost:3000)
npm start
```

The app auto-detects the environment: if the host is not `hr.insightgenie.ai`, it points to `localhost:9718` for the API.

### Build for Production

```bash
npm run build
```

Output goes to the `build/` folder, ready for static hosting.

### Run Tests

```bash
npm test
```
